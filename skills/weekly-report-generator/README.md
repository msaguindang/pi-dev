# Weekly Report Generator

An Opencode skill that automates the creation of the Weekly Report.

It scans the Obsidian vault for daily work logs matching a specific ISO week, extracts the raw technical notes and Discord EOD updates, synthesizes them into a management-friendly executive summary, and generates a `.docx` version using `pandoc`.

## Usage
Just ask the agent: "Generate the weekly report for this week" or "Write the report for last week."
